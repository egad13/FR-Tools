Welcome!

This is a collection of tools to help expedite some tedious tasks for players of Flight Rising. Try one of the navigation links above to get started!

Having Issues?

If one of the tools isn't working properly, it's possible it got updated and your browser didn't get the memo. Try clearing your browser cache and then see if it works.

If that still doesn't resolve the issue, and you're on a modern, updated browser, there might be a bonafide bug. Please either open an issue on Github or contact Kay on Flight Rising - I'll try to track down the problem as soon as possible!

For Developers

The source code for the tools is Here. The documentation for that source code is Here.

You may be particularly interested in the frdata module, which provides data on Flight Rising's colours, breeds, genes, and more, as well as a suite of useful functions for asking questions about and comparing that data. Feel free to use that (with credit!) in your own projects.

Because this project uses ES6 Modules, you can't locally test it just by opening the files in a browser. You'll need to run a local HTTP server. MDN has a guide on how to do that.
